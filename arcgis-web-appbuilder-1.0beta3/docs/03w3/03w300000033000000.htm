Get started

Install

Download Web AppBuilder for ArcGIS ZIP file to your local drive and unzip it.

Launch Web AppBuilder

On Windows, double click the startup.bat file inside the unzipped folder. Web AppBuilder starts in your default browser. Web AppBuilder runns on the top of Node.js with default port of 3344. It is not necessary to install Node.js separately as it is contained in the ZIP file.

On other platforms, open the command prompt window, navigate to the server directory, then type: node server.js. Once the Node.js starts, launch Web AppBuilder in the browser with http://[yourmachinename]:3344/webappbuilder. 3344 is the default port for Node.js.

If you want to run Web AppBuilder with additional parameters or different port , follow the instructions below:

  1. Open a command window and navigate to the server directory of Web AppBuilder installation.
    • On 64bit Windows, type: node_x64.exe server.js [-port=value] [-proxy=value]
    • On 32bit Windows, type: node_x32.exe server.js [-port=value] [-proxy=value]
    • On other platforms, type: server.js [-port=value] [-proxy=value]
    Note:

    -port specifies the port you want Node.js to run on, and -proxy specifies the proxy URL if needed.

  2. Launch Web AppBuilder in the browser with http://[yourmachinename]:[port]/webappbuilder, where [port] is the port number specified in step 1.
    Tip:

    If you have problems to launch Web AppBuilder, refer to FAQs for more information.

Log in with named user account

To access Web AppBuilder for ArcGIS, you must have an ArcGIS Online organization account or a Portal for ArcGIS account.

Subscription:

Sign up for the ArcGIS Online Trial account at http://www.esri.com/software/arcgis/arcgisonline/evaluate.

  1. Specify your URL for ArcGIS Online organization or Portal for ArcGIS.
  2. Since this is the first time you've used Web Appbuilder with the above specified URL, you will be asked to log in to generate an AppID.
    Tip:

    AppID is required by OAuth2 authentication. See Add apps for more details. If you have problems to log in, refer to FAQs for more information.

Create your first app

Do the following steps to create a basic web application:

  1. Click the Create New button on the home page.
  2. Provide the app name and description, and click OK.
  3. Accept the default theme, map, and widgets.
  4. Click the Attributes tab. Change the title, subtitle, and logo of the application.
  5. Click Save to save your changes.
  6. Click Previews to view the app in devices.
  7. Click the Web AppBuilder or Icon on the upper left corner to return to the home page.
  8. Highlight the application in the list and click the Launch button to view it in the web browser.
