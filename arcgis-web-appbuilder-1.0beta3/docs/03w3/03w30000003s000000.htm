Coordinate widget

The Coordinate widget enables x,y coordinate values to appear in your application. With the default coordinate system of the webmap, the coordinate values change dynamically as the mouse pointer moves to different locations on the map. Multiple spatial references can be configured using the widget's configuration window, and the coordinate values are based on the active spatial reference you selected when the application starts. When the widget is configured to use multiple spatial reference values, hover over the widget and select what coordinate system to display, then click on a location to display values based on the selected coordinate system.

Setting the Coordinate widget

By default, the Coordinate widget displays coordinates in WGS_1984_Web_Mercator_Auxiliary_Sphere (wkid 3857). Displaying coordinates in other projections or datums can be done by adding spatial references to the Coordinate widget using its configuration window.

Steps:
  1. Hover the mouse over the Coordinate widget and click the small eye icon to show or hide this widget in your application .

    By default, the widgets included with a theme are visible.

  2. Hover the mouse over the Coordinate widget and click the small edit icon .

    This opens the configuration window for this widget.

  3. Click Add Output Coordinates System.

    The add output coordinates system window displays.

  4. Provide a valid WKID in the Output Coordinate System WKID box, select a display unit, and provide a valid WKID in the Datum Transformation WKID box if the datum of output coordinate system is different form the default.
  5. Click the OK button.

    The above defined spatial reference is added to the list. Repeat the above steps to add another spatial reference if necessary.

  6. Click OK.

    The widget setting window closes.

Using the Coordinate widget

When the application starts, the coordinates are automatically enabled for the default spatial reference and any additional spatial reference specified in the configuration. Hover the mouse over the Coordinate widget on the map to display a list of spatial references. When you select one from the list, it becomes the active spatial reference. With the default Web Mercator projection, when you move the mouse in different locations on the map, the coordinates display.

With other spatial references, click a point on the map to display the coordinates.
