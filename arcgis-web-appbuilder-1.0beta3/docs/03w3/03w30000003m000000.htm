Create new app

Web AppBuilder for ArcGIS enables a quick and easy way to build web mapping applications. It makes the process seamless without requiring programming or editing of configuration files. Creating a new application with Web AppBuilder involves the following steps:

  • Log in with named user account
  • Choose the look-and-feel of the application
  • Specify the map by choosing a web map
  • Configure the app's functionality
  • Brand the application
  • Preview the application on different platforms

Create a basic web application

Do the following steps to create a basic web application:

  1. Log in Web AppBuilder with your ArcGIS Online organization account or ArcGIS for Portal account.
  2. On the home page, click the Create New button.
  3. Provide the app name and description, and click OK.
  4. Accept the default theme, map, and widgets.
  5. Click the Attributes tab, change the title, subtitle, and logo of the application.
  6. Click Save to save your changes.
  7. Click Previews to view the app in devices.
  8. Click Web AppBuilder or Icon on the upper left corner to return to the home page.
  9. Highlight the application in the list and click the Launch button to view it in the web browser.

More configuration options can be defined by accessing the tabs including Themes, Map, Widgets, and Attributes.

Create your app by configuring the starter application

Web AppBuilder provides an option to create a web app by configuring the starter app (the stemapp folder). For details, see JSON configuration reference.
