Attributes

The Attributes tab allows you to perform basic customization of your application's banner.

The following items can be added to the banner:

  • Logo—Click the logo icon to open the file browser and select an image file.
  • Title
  • Subtitle
  • One or more hyperlinks

When the application starts, if necessary, expand your browser to see all of the above items, especially when the text size is large.

Click the Add New Link button to add a hyperlink to the banner and provide a name and valid URL for it. Edit or remove a link by clicking the corresponding icons next to it.

Caution:

Only maximum of four links are allowed to add.
