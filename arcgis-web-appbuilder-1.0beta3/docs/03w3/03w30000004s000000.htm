Export as template

The Web AppBuilder enables you to easily create custom web app templates for your organization by exporting the app. Your custom template must be hosted locally.

To do this, start with the app on the Home page:

Steps:
  1. Select the app. Move the mouse over the more icon , and click Export As Template.
  2. On the left panel of the Set Configurable Parameters window, select the parameters that you will let users customize, for example logo, and click the icon in line with the parameter. The logo parameter will be added in the right panel of the window.

    Configurable parameters such as title and subtitle are added by default. therefore there is no icon associated with them.

  3. Expand other parameters, for example widgetPool, to add more configurable parameters.
  4. Click Save to save the configurable parameters in the app.

    The saved configurable parameters will reload when the Set Configurable Parameters window opens next time. They are persisted in the app config file.

    For more information about configurable parameters, refer to Add configurable parameters to templates .

  5. Click Export. In the Export as Web App Template window, provide the template information with the Title, Tags and Description. Click OK.
  6. Depending on whether you have permission to share your exported template to the template group specified in your organization, you may get one of these messages.

    • If you don't, the message is
  7. Click OK. In either case, your template will be added as an item in My Content. Navigate to My Content to see your custom template.

    The Export As Template will not only create a template as an app item under your account, but also try to share it with the “Web App Templates” group specified in your organization. If you do not have privilege to share the item with the specified group, a warning message will display.

  8. Since the template has been exported successfully, click Cancel to close the Set Configurable Parameters window.
  9. Share the item to the group being used for the templates if you want to use the configurable template in your organization's web template gallery or group template gallery. Then the administrator of your organization can configure the map viewer or configure groups to use the group that includes your template.
  10. Publish a map with your template

    Refer to Create apps from maps for details.
