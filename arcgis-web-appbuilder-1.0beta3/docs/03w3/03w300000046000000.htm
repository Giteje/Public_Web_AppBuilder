Widgets

The Widgets tab allows you to configure the functionality of an application by selecting widgets from the widget collection. The initial set of widgets may vary from theme to theme as each theme has its own pre-configured set of widgets. The pre-configured widgets for a theme display when the Widgets tab is activated. Off-panel widgets are currently available as part of a theme, and can be turned on or off but cannot be removed or added from the Set controlled widgets list. In-panel widgets can be added from the Set controlled widgets list or removed from your application.

Besides the pre-configured widgets within a theme, you can always add additional widgets to the application by searching the controlled widgets collection. There are two ways to add in-panel widgets from the widget collection. One is by setting the controlled widgets and another is by placeholder. When a widget is added to the application, configure it by clicking the small edit icon for the widget.

Clicking Set the widgets in this controller opens the page for in-panel widgets. If a theme has any in-panel widgets, they show on this page. Clicking the Click here to add widget icon opens the widget collection window that allows you to select and add a widget to the application.

Selected widgets highlight in blue boxes. Multiple widgets can be selected. If you have custom widgets, they also show in the widget collection.

Clicking placeholder labeled by a number, opens the widget collection window. Select one widget for each placeholder.

When the widget configuration completes, click the Save.
