Edit or upload an app

Edit an application

All existing apps list on the Web AppBuilder for ArcGIS home page. Clicking the Edit icon next to an application opens it for editing.

Upload an application

Web AppBuilder also supports uploading an existing application that may be created by a different user. To do so, click the Upload button to open the file browser, point to the ZIP file of an existing application, then click open. An application with the ZIP file name is created in the Builder. If an application with the same name already exists, a suffix is automatically added to the uploaded application name.
