Basemap Gallery widget

This widget presents a gallery of basemaps and allows you to select one from the gallery as the application's basemap. The basemaps can be user-defined or from your ArcGIS organization. However, all basemaps added to the gallery must be in the same projection. If you are displaying basemaps from your organization, additional custom basemaps must be in the Web Mercator projection (wkid: 102100, 102113, and 3857).

Unlike the theme widgets whose icons cannot be changed, this widget is part of the widgets controlled by Header Controller and its icon can be changed by using your own image file.

Setting the Basemap Gallery widget

Steps:
  1. Hover the mouse over the Basemap Gallery widget and click the small edit icon .

    This opens the configuration window for this widget.

  2. Optionally, click change widget icon if you want to replace the default icon for this widget.

    A file explorer window opens allowing you to browse to a local image file to use as the widget icon.

  3. Check or uncheck the Show Portal Basemaps option.

    When this option is checked, portal basemaps display in the gallery. You can still add your custom basemaps to the gallery but they need to be in the same projection as the portal basemaps.

  4. Click Click to add a new basemap.

    The Add Basemap window displays.

  5. Provide a title, the URL of your map service, and click the thumbnail image to replace it with your own image file.
  6. Click Add Basemap to add additional URL of a map service to the basemap .

    A single basemap can contain multiple maps, but they need to be in a same projection.

  7. Click OK to close this window.

    You basemap is added to the list of basemaps with the selected thumbnail.

Using the Basemap Gallery widget

Clicking the Basemap Gallery widget displays all the basemaps configured for this widget. Clicking one of the basemap thumbnails sets it as the active basemap for the application. Click the X button in the upper right corner of the Basemap Gallery window to close it. When ArcGIS basemaps are in the gallery, selecting other basemaps whose projection is different from Web Mercator sets the basemap to blank.
