Geoprocessing widget

The Geoprocessing widget provides a dynamic user interface to execute a geoprocessing task. It provides the ability to configure symbology for input and output layers, as well as overriding default values for parameters.

Configuring the Geoprocessing widget

Click the edit widget icon to open the configuration properties dialog box.

Steps:
  1. Click the Set button to enter the geoprocessing task URL in the Task URLfield.
  2. The list of Input and Output parameters for the task populates.
  3. To configure each Input parameter, click to make it active. The default values, defined by the published geoprocessing service, populate in the Label, Tooltip, and Default Value areas.
  4. Check Visible to display the parameter on the widget interface.
  5. The content of the Default Value region varies, based on the parameter type. For details, refer to Geoprocessing parameter types.
  6. Configure the Renderer for the Output parameter using the symbol selection tools.
  7. Select the fields to display on the Popup window for output features. Enable the popup by checking Enable popup. Check the Visible column to display the field in the window when a feature is clicked. Reorder fields by clicking the up and down arrows next to the field.
  8. Control the order of the layers (inputs, output) by clicking the up and down arrows in the Layer order section.
  9. Check the boxes to override the output layer symbology, or to add the output layer to the map's operational layers in the Options section.
  10. Click OK to close the dialog box.

Using the Geoprocessing widget

Steps:
  1. Click the widget icon to open the Ggeoprocessing widget.
  2. Enter a value for each of the input parameters. In this example, the drive time analysis geoprocessing task from ArcGIS Online is consumed. The task requires one or more starting locations and a set of space-delimited drive times used to calculate possible distances over time.
  3. Most geoprocessing tasks require input using the map to draw geometries to provide as input for the geoprocessing task. Click the symbol icon to activate the drawing tool and draw a feature on the map. To remove all features and start again, click Clear.
  4. Once all required input parameters (annotated by an asterisk next to the name) have values, click Execute to submit the task. The Output tab is selected and displays a progress indicator until execution completes.
  5. Upon completion, the geoprocessing results display on the map. Click Output to view their attributes.
  6. Click Clear on the Output tab to clear the geoprocessing results from the map.
